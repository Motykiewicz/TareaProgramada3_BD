Usuario administrador

Resumen rápido

Desde este panel el administrador puede revisar el estado general del sistema municipal, consultar reportes y realizar pruebas de facturación, pagos y morosidad.

Esta vista está pensada principalmente para pruebas de la tarea:

  • Ver información general del sistema.
  • Impersonar a un usuario para ver su vista de cliente.
  • Consultar reportes y pruebas que pide la tarea.
  • Probar facturación, pagos y morosidad del sistema.

Usuarios del sistema

Seleccione un usuario para ver el portal como si fuera ese cliente. Esta funcionalidad se usa para probar la consulta web del cliente sin necesidad de conocer su contraseña.
